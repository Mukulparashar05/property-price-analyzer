# House Price Presentation Design

**Goal:** Create a short, general-explanation PowerPoint deck about the house price prediction project.

## Audience

General audience with no deep machine learning background.

## Slide Plan

1. **Title**
   House Price Prediction Project with a short subtitle explaining the project context.
2. **Objectives**
   Explain the project goals in simple bullets.
3. **Dataset Overview**
   Summarize the California housing dataset, the target column, and major input features.
4. **ML Model**
   Describe the notebook's machine learning approach, including preprocessing and candidate regression models.
5. **Architecture Diagram**
   Show the data flow from CSV input through preprocessing, splitting, training, evaluation, and prediction.

## Design Choices

- Keep slides clean and readable with a simple color palette.
- Prefer short bullets over dense paragraphs.
- Use a diagram slide to make the workflow easier to understand visually.
- Keep content aligned with the actual state of the project: exploratory notebook with planned model pipeline.

## Output

A real `.pptx` file saved in the project workspace and editable in PowerPoint-compatible software.
